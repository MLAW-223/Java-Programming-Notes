# Java ArrayLists

## What is an ArrayList?
An ArrayList in Java is a dynamic, resizable array that stores objects. Unlike traditional arrays, which have a fixed size upon creation, ArrayLists can grow or shrink as needed.

**Resizable:** ArrayLists can change their size dynamically.

**Object Storage:** They store objects. If you need to store primitive data types (like `int`, `double`, `boolean`), you'll use their corresponding wrapper classes (`Integer`, `Double`, `Boolean`) and rely on Java's autoboxing feature.

## Creating an ArrayList
To create an ArrayList, you need to:

1. **Import the ArrayList class:**
   ```java
   import java.util.ArrayList;
   ```

2. **Declare and instantiate an ArrayList:**
   ```java
   ArrayList<DataType> listName = new ArrayList<>();
   ```
   - `DataType`: Specifies the type of objects the ArrayList will store. For primitives, use their wrapper class (e.g., `Integer`, `String`, `Double`).
   - `listName`: The name you give to your ArrayList variable.
   - The diamond operator `<>` infers the data type for the second ArrayList declaration, so you don't need to repeat the `DataType` inside it.

**Example:**
```java
// An ArrayList to store integers
ArrayList<Integer> numbers = new ArrayList<>();

// An ArrayList to store strings
ArrayList<String> fruits = new ArrayList<>();
```

## Common ArrayList Methods

### Adding Elements
**`add(DataType element)`:** Appends an element to the end of the ArrayList.

**`add(int index, DataType element)`:** Inserts an element at a specific index, shifting existing elements to the right.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");      // fruits is now ["Apple"]
fruits.add("Banana");     // fruits is now ["Apple", "Banana"]
fruits.add(0, "Orange");  // fruits is now ["Orange", "Apple", "Banana"]
```

### Removing Elements
**`remove(int index)`:** Removes the element at the specified index.

**`remove(Object o)`:** Removes the first occurrence of the specified element.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Orange");

fruits.remove(0);        // Removes "Apple". fruits is now ["Banana", "Orange"]
fruits.remove("Banana"); // Removes "Banana". fruits is now ["Orange"]
```

### Setting (Updating) Elements
**`set(int index, DataType element)`:** Replaces the element at the specified index with a new element.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");

fruits.set(0, "Pineapple"); // Replaces "Apple" at index 0 with "Pineapple".
                           // fruits is now ["Pineapple", "Banana"]
```

### Getting Elements
**`get(int index)`:** Returns the element at the specified index.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");

String firstFruit = fruits.get(0); // firstFruit will be "Apple"
String secondFruit = fruits.get(1); // secondFruit will be "Banana"
```

### Getting the Size
**`size()`:** Returns the number of elements currently in the ArrayList.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Orange");

int numberOfFruits = fruits.size(); // numberOfFruits will be 3
```

### Sorting an ArrayList
To sort an ArrayList, you can use the `Collections` class.

1. **Import the Collections class:**
   ```java
   import java.util.Collections;
   ```

2. **Use the `sort()` method:**
   ```java
   Collections.sort(listName);
   ```
   This method sorts the elements in ascending (natural) order.

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Banana");
fruits.add("Apple");
fruits.add("Orange");
fruits.add("Coconut");

Collections.sort(fruits); // Sorts the list alphabetically.
                          // fruits is now ["Apple", "Banana", "Coconut", "Orange"]
```

### Iterating Through an ArrayList
You can iterate through the elements of an ArrayList using an enhanced for loop (for-each loop).

**Example:**
```java
ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
fruits.add("Coconut");
fruits.add("Orange");

for (String fruit : fruits) {
    System.out.println(fruit);
}
```

This will print each fruit on a new line:
```
Apple
Banana
Coconut
Orange
```

## Exercise Example: Storing User-Input Food Items
This example demonstrates how to create an ArrayList to store food items entered by the user.

**Steps:**
1. Import necessary classes: `ArrayList` and `Scanner`.
2. Create a `Scanner` object to read user input.
3. Create an `ArrayList` to store `String` elements (food names).
4. Prompt the user for the number of food items they want to enter.
5. Read the integer representing the number of food items.
6. Clear the input buffer after reading the integer.
7. Use a for loop to iterate the specified number of times.
8. Inside the loop:
   - Prompt the user to enter each food item.
   - Read the food item (as a String).
   - Add the food item to the ArrayList.
9. After the loop, print the entire ArrayList of foods.
10. Close the Scanner.

**Code:**
```java
import java.util.ArrayList;
import java.util.Scanner;

public class FoodListCreator {
    public static void main(String[] args) {

        // 1. Import and create Scanner
        Scanner scanner = new Scanner(System.in);

        // 2. Create ArrayList for foods
        ArrayList<String> foods = new ArrayList<>();

        // 3. Prompt user for number of items
        System.out.print("Enter the number of food items you would like to store: ");
        int numberOfFoodItems = scanner.nextInt();

        // 4. Clear input buffer
        scanner.nextLine();

        // 5. Loop to get food items from user
        for (int i = 1; i <= numberOfFoodItems; i++) {
            System.out.print("Enter food number " + i + ": ");
            String foodItem = scanner.nextLine();
            foods.add(foodItem); // Add the entered food to the ArrayList
        }

        // 6. Print the ArrayList
        System.out.println("\nYour list of foods:");
        System.out.println(foods);

        // 7. Close the scanner
        scanner.close();
    }
}
```

**Example Run:**
```
Enter the number of food items you would like to store: 3
Enter food number 1: Pizza
Enter food number 2: Hot Dog
Enter food number 3: Hamburger

Your list of foods:
[Pizza, Hot Dog, Hamburger]
```
