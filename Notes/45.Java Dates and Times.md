# Java Dates and Times

## Introduction to Date and Time in Java
Java's `java.time` package provides a comprehensive API for working with dates and times. This guide covers the fundamental classes and operations demonstrated in the video.

## Core Date and Time Classes

### LocalDate
Represents a date without a time-of-day or time zone.

**Getting the current date:**
```java
LocalDate date = LocalDate.now();
System.out.println(date);
```

**Creating a specific date:**
```java
// Year, Month (1-12), Day
LocalDate specificDate = LocalDate.of(2024, 12, 25); // December 25, 2024
System.out.println(specificDate);
```

### LocalTime
Represents a time without a date or time zone.

**Getting the current time:**
```java
LocalTime time = LocalTime.now();
System.out.println(time);
```

**Creating a specific time:**
```java
// Hour (0-23), Minute, Second
LocalTime specificTime = LocalTime.of(12, 0, 0); // 12:00:00 (noon)
System.out.println(specificTime);
```

### LocalDateTime
Represents a date and time without a time zone. This is a combination of `LocalDate` and `LocalTime`.

**Getting the current date and time:**
```java
LocalDateTime dateTime = LocalDateTime.now();
System.out.println(dateTime);
```

**Creating a specific date and time:**
```java
// Year, Month, Day, Hour, Minute, Second
LocalDateTime specificDateTime = LocalDateTime.of(2024, 12, 25, 12, 0, 0); // Christmas Day, 12:00:00
System.out.println(specificDateTime);
```

### Instant
Represents a point in time on the time-line, typically in UTC (Coordinated Universal Time). Useful for machine-readable timestamps.

**Getting the current UTC timestamp:**
```java
Instant instant = Instant.now();
System.out.println(instant);
```

## Formatting Dates and Times
The `DateTimeFormatter` class allows you to define custom patterns for displaying date and time objects.

**Creating a custom formatter:**
```java
// Define the pattern using specific symbols
// dd: day of month (e.g., 17)
// MM: month of year (e.g., 12)
// yyyy: year (e.g., 2024)
// HH: hour of day (0-23)
// mm: minute of hour (e.g., 05)
// ss: second of minute (e.g., 09)
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm:ss");
```

**Formatting a LocalDateTime object:**
```java
LocalDateTime dateTime = LocalDateTime.now();
String formattedDateTime = dateTime.format(formatter);
System.out.println(formattedDateTime);
```

### Common Pattern Letters for DateTimeFormatter
| Pattern | Description | Example Output |
|---------|-------------|----------------|
| d | Day of month (1-31) | 5 |
| dd | Day of month (01-31) | 05 |
| M | Month of year (1-12) | 12 |
| MM | Month of year (01-12) | 12 |
| MMM | Month of year (text, short) | Dec |
| MMMM | Month of year (text, full) | December |
| y | Year | 2024 |
| yy | Year (last two digits) | 24 |
| yyyy | Year (four digits) | 2024 |
| H | Hour of day (0-23) | 9 |
| HH | Hour of day (00-23) | 09 |
| h | Hour of day (1-12) | 9 |
| hh | Hour of day (01-12) | 09 |
| m | Minute of hour (0-59) | 5 |
| mm | Minute of hour (00-59) | 05 |
| s | Second of minute (0-59) | 9 |
| ss | Second of minute (00-59) | 09 |
| S | Fraction of second (milliseconds) | 123 |

## Comparing Dates and Times
The `LocalDateTime` class provides methods to compare two date-time objects.

**`isBefore(ChronoLocalDateTime<?> other)`:** Returns `true` if this date-time is before the specified date-time.

**`isAfter(ChronoLocalDateTime<?> other)`:** Returns `true` if this date-time is after the specified date-time.

**`isEqual(ChronoLocalDateTime<?> other)`:** Returns `true` if this date-time is equal to the specified date-time.

### Example Comparison
```java
LocalDateTime date1 = LocalDateTime.of(2024, 12, 25, 12, 0, 0); // Christmas Day, 2024
LocalDateTime date2 = LocalDateTime.of(2025, 1, 1, 0, 0, 0);   // New Year's Day, 2025

if (date1.isBefore(date2)) {
    System.out.println(date1 + " is earlier than " + date2);
} else if (date1.isAfter(date2)) {
    System.out.println(date1 + " is later than " + date2);
} else if (date1.isEqual(date2)) {
    System.out.println(date1 + " is equal to " + date2);
}
```
