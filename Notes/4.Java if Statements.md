# Java if Statements

## What is an if Statement?

An if statement in Java is a control flow statement that allows you to execute a block of code only if a specified condition is true.

---

## Basic if Statement

The fundamental structure of an if statement is:

```java
if (condition) {
    // Code to execute if the condition is true
}
```

- **condition**: This is a Boolean expression that evaluates to either `true` or `false`
- The code within the curly braces `{}` is executed only when the condition evaluates to `true`

**Example:**

```java
int age = 20;

if (age >= 18) {
    System.out.println("You are an adult.");
}
```

In this example, if the `age` variable is greater than or equal to 18, the message "You are an adult." will be printed. If age were less than 18, this line of code would be skipped.

---

## if-else Statement

An if-else statement provides an alternative block of code to execute when the if condition is false.

```java
if (condition) {
    // Code to execute if the condition is true
} else {
    // Code to execute if the condition is false
}
```

**Example:**

```java
int age = 12;

if (age >= 18) {
    System.out.println("You are an adult.");
} else {
    System.out.println("You are a child.");
}
```

Here, since age is 12 (which is not greater than or equal to 18), the else block will be executed, printing "You are a child."

---

## if-else if-else Statement

You can chain multiple conditions using `else if`. This allows you to check several conditions in sequence. The `else` block at the end acts as a default if none of the preceding `if` or `else if` conditions are true.

```java
if (condition1) {
    // Code if condition1 is true
} else if (condition2) {
    // Code if condition1 is false and condition2 is true
} else if (condition3) {
    // Code if condition1 and condition2 are false, and condition3 is true
} else {
    // Code if all previous conditions are false
}
```

### Key Points:

- The conditions are evaluated from top to bottom
- As soon as a condition evaluates to `true`, its corresponding code block is executed, and the rest of the `else if` and `else` blocks are skipped
- The `else` block is optional

**Example:**

```java
int age = 70;

if (age >= 65) {
    System.out.println("You are a senior.");
} else if (age >= 18) {
    System.out.println("You are an adult.");
} else if (age >= 0) {
    System.out.println("You are a child.");
} else {
    System.out.println("You haven't been born yet.");
}
```

In this scenario, age is 70:
- The first condition `age >= 65` is `true`
- The code within the first if block ("You are a senior.") is executed
- The remaining `else if` and `else` blocks are skipped

---

### Order Matters:

Consider the previous example with `age = 20`:

```java
int age = 20;

if (age >= 65) { // False
    System.out.println("You are a senior.");
} else if (age >= 18) { // True
    System.out.println("You are an adult.");
} else if (age >= 0) {
    System.out.println("You are a child.");
} else {
    System.out.println("You haven't been born yet.");
}
```

Here, the first condition is false. The second condition `age >= 18` is true, so "You are an adult." is printed, and the rest are skipped.

**If the order were reversed:**

```java
int age = 20;

if (age >= 18) { // True
    System.out.println("You are an adult.");
} else if (age >= 65) { // This condition will never be checked if the first is true
    System.out.println("You are a senior.");
} else {
    System.out.println("You are a child.");
}
```

**Important:** If age was 70 and the `if (age >= 18)` came first, the output would be "You are an adult.", even though "You are a senior." might also be a correct description. The positioning of your `if` and `else if` statements is crucial.

## Comparison Operators

These operators are used to form the conditions in if statements:

- **`>`**: Greater than
- **`<`**: Less than
- **`>=`**: Greater than or equal to
- **`<=`**: Less than or equal to
- **`==`**: Equal to (use double equals for comparison)
- **`!=`**: Not equal to

### Important Note on Equality:

- Use `==` for comparing values (e.g., `age == 0`)
- Use `=` for assigning values to variables (e.g., `age = 0`)
- Using a single equals `=` in a condition will result in a compilation error because Java interprets it as an assignment attempt

---

## Working with Strings

When comparing strings, you **cannot** use the `==` operator to check for equality of content. Instead, you should use the `.equals()` method.

```java
String name = "Alice";

if (name.equals("Alice")) {
    System.out.println("Hello, Alice!");
}
```

To check if a string is empty, you can use the `.isEmpty()` method:

```java
String userName = ""; // Or obtained from user input

if (userName.isEmpty()) {
    System.out.println("You did not enter your name.");
} else {
    System.out.println("Hello, " + userName + "!");
}
```

## User Input with Scanner

To get input from the user, you can use the `Scanner` class.

1. **Import the Scanner class:**
   ```java
   import java.util.Scanner;
   ```

2. **Create a Scanner object:**
   ```java
   Scanner scanner = new Scanner(System.in);
   ```

3. **Read user input:**
   - For integers: `int value = scanner.nextInt();`
   - For strings (a single word): `String word = scanner.next();`
   - For strings (an entire line): `String line = scanner.nextLine();`
   - For booleans: `boolean boolValue = scanner.nextBoolean();`

4. **Close the Scanner when done:**
   ```java
   scanner.close();
   ```

**Example using Scanner:**

```java
import java.util.Scanner;

public class AgeChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Enter your age: ");
        int age = scanner.nextInt();

        if (age >= 18) {
            System.out.println("You are an adult.");
        } else {
            System.out.println("You are a child.");
        }

        scanner.close();
    }
}
```
