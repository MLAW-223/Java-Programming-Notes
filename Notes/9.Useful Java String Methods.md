# Useful Java String Methods

## Introduction to Java Strings

In Java, a String is an object that represents a sequence of characters. Strings are **immutable**, meaning once a String object is created, its content cannot be changed. Any operation that appears to modify a string actually creates a **new** String object.

---

## String Methods

The String class in Java provides numerous methods for manipulating and querying string objects. To use these methods, you typically call them on a String variable using the dot (.) operator.

---

## Getting String Information

### `length()`
Returns the number of characters in the string.

**Syntax:** `int length = stringVariable.length();`

**Example:**
```java
String name = "Bro Code";
int length = name.length(); // length will be 8
System.out.println(length);
```

### `charAt(int index)`
Returns the character at the specified index. Indices are 0-based, meaning the first character is at index 0.

**Syntax:** `char character = stringVariable.charAt(index);`

**Example:**
```java
String name = "Bro Code";
char firstChar = name.charAt(0); // firstChar will be 'B'
char thirdChar = name.charAt(2); // thirdChar will be 'o'
System.out.println(firstChar);
System.out.println(thirdChar);
```

### `indexOf(String str)`
Returns the index of the first occurrence of the specified substring. Returns -1 if the substring is not found.

**Syntax:** `int index = stringVariable.indexOf(substring);`

**Example:**
```java
String name = "Bro Code";
int spaceIndex = name.indexOf(" "); // spaceIndex will be 3 (0-based index)
int oIndex = name.indexOf("o");   // oIndex will be 2 (first 'o')
System.out.println(spaceIndex);
System.out.println(oIndex);
```

### `lastIndexOf(String str)`
Returns the index of the last occurrence of the specified substring. Returns -1 if the substring is not found.

**Syntax:** `int index = stringVariable.lastIndexOf(substring);`

**Example:**
```java
String name = "Bro Code";
int lastOIndex = name.lastIndexOf("o"); // lastOIndex will be 5
System.out.println(lastOIndex);
```

## Modifying String Content (Creating New Strings)

These methods do not change the original string but return a new string with the modifications.

### `toUpperCase()`
Converts all characters in the string to uppercase.

**Syntax:** `String upperCaseString = stringVariable.toUpperCase();`

**Example:**
```java
String name = "Bro Code";
String upperName = name.toUpperCase(); // upperName will be "BRO CODE"
System.out.println(upperName);
```

### `toLowerCase()`
Converts all characters in the string to lowercase.

**Syntax:** `String lowerCaseString = stringVariable.toLowerCase();`

**Example:**
```java
String name = "Bro Code";
String lowerName = name.toLowerCase(); // lowerName will be "bro code"
System.out.println(lowerName);
```

### `trim()`
Removes leading and trailing whitespace from the string.

**Syntax:** `String trimmedString = stringVariable.trim();`

**Example:**
```java
String nameWithSpaces = "   Bro Code   ";
String trimmedName = nameWithSpaces.trim(); // trimmedName will be "Bro Code"
System.out.println(trimmedName);
```

### `replace(char oldChar, char newChar)`
Replaces all occurrences of a specified character with another character.

**Syntax:** `String replacedString = stringVariable.replace(oldChar, newChar);`

**Example:**
```java
String name = "Bro Code";
String replacedName = name.replace('o', 'a'); // replacedName will be "Bra Cade"
System.out.println(replacedName);
```

### `replace(CharSequence target, CharSequence replacement)`
Replaces all occurrences of a target character sequence with a replacement character sequence.

**Syntax:** `String replacedString = stringVariable.replace(targetSequence, replacementSequence);`

**Example:**
```java
String message = "Hello World";
String newMessage = message.replace("World", "Java"); // newMessage will be "Hello Java"
System.out.println(newMessage);
```

## Boolean String Checks

These methods return a boolean value (`true` or `false`).

### `isEmpty()`
Returns `true` if the string has a length of 0, and `false` otherwise.

**Syntax:** `boolean isEmpty = stringVariable.isEmpty();`

**Example:**
```java
String emptyString = "";
String nonEmptyString = "Hello";

System.out.println(emptyString.isEmpty()); // Output: true
System.out.println(nonEmptyString.isEmpty()); // Output: false
```

This is often used in conditional statements:
```java
String userName = ""; // Or obtained from user input
if (userName.isEmpty()) {
    System.out.println("Username cannot be empty.");
} else {
    System.out.println("Welcome, " + userName + "!");
}
```

### `contains(CharSequence s)`
Returns `true` if the string contains the specified sequence of characters, and `false` otherwise.

**Syntax:** `boolean contains = stringVariable.contains(sequence);`

**Example:**
```java
String email = "user@example.com";
boolean hasAtSymbol = email.contains("@"); // hasAtSymbol will be true
System.out.println(hasAtSymbol);

String message = "Java programming";
boolean hasSpace = message.contains(" "); // hasSpace will be true
System.out.println(hasSpace);
```

### `equals(Object anObject)`
Compares this string to the specified object. Returns `true` if and only if the argument is not null and is a String object that represents the same sequence of characters as this object. This method is **case-sensitive**.

**Syntax:** `boolean isEqual = stringVariable1.equals(stringVariable2);`

**Example:**
```java
String passwordAttempt = "password123";
String correctPassword = "password123";
String wrongPassword = "Password123";

System.out.println(passwordAttempt.equals(correctPassword)); // Output: true
System.out.println(passwordAttempt.equals(wrongPassword));   // Output: false (case-sensitive)
```

### `equalsIgnoreCase(String anotherString)`
Compares this string to another string, ignoring case considerations. Returns `true` if and only if the argument is not null and is a String object that represents the same sequence of characters as this object, ignoring case.

**Syntax:** `boolean isEqualIgnoreCase = stringVariable1.equalsIgnoreCase(stringVariable2);`

**Example:**
```java
String inputPassword = "Password123";
String storedPassword = "password123";

System.out.println(inputPassword.equalsIgnoreCase(storedPassword)); // Output: true
```

## Activity 1: Text Analyzer
Create a program that analyzes text input and provides statistics:

**Requirements:**
1. Count total characters (including spaces)
2. Count total words (assume words are separated by spaces)
3. Find and display the most common letter (case-insensitive)
4. Check if the text is a palindrome (reads the same forwards and backwards, ignoring case and spaces)
5. Calculate the average word length

**Sample Input:** "A man a plan a canal Panama"
**Expected Output:**
- Total characters: 30
- Total words: 7
- Most common letter: 'a'
- Is palindrome: true
- Average word length: 3.4

**Challenge:** Handle punctuation correctly (ignore it when analyzing letters).

**Username Rules:**
1. Must be between 6-12 characters
2. Must contain only letters, numbers, and underscores
3. Must start with a letter
4. Cannot end with an underscore

**Password Rules:**
1. Must be at least 8 characters
2. Must contain at least one uppercase letter
3. Must contain at least one lowercase letter
4. Must contain at least one number
5. Must contain at least one special character (!@#$%^&*)
6. Cannot contain spaces
7. Cannot contain the username (case-insensitive)

**Your program should:**
- Ask for username and password
- Validate each against the rules
- Provide specific feedback for each rule that fails
- Only accept the credentials when all rules pass
