# Java printf() Formatting

## Introduction to printf()

The `printf()` method in Java is a powerful tool for formatting output. It allows for more control over how variables are displayed compared to `print()` or `println()`. The general structure of a `printf()` statement is:

```java
System.out.printf("format string", arguments);
```

The **"format string"** contains literal text and **"format specifiers"** which act as placeholders for the arguments.

---

## Format Specifier Structure

A format specifier follows this pattern:

```
% [flags] [width] [.precision] [specifier-character]
```

---

## Specifier Characters

These characters indicate the data type of the argument to be formatted.

- **`%s`**: String
- **`%c`**: Character
- **`%d`**: Integer (decimal)
- **`%f`**: Floating-point numbers (double, float)
- **`%b`**: Boolean

### Example Usage of Specifier Characters

```java
String name = "Spongebob";
char firstLetter = 'S';
int age = 30;
double height = 60.5;
boolean isEmployed = true;

System.out.printf("Hello %s\n", name);
// Output: Hello Spongebob

System.out.printf("Your name starts with a %c\n", firstLetter);
// Output: Your name starts with a S

System.out.printf("You are %d years old\n", age);
// Output: You are 30 years old

System.out.printf("You are %f inches tall\n", height);
// Output: You are 60.500000 inches tall

System.out.printf("Employed: %b\n", isEmployed);
// Output: Employed: true

// Multiple arguments
System.out.printf("%s is %d years old\n", name, age);
// Output: Spongebob is 30 years old
```

---

## Precision

The **precision** part of the format specifier controls the number of digits displayed after the decimal point for floating-point numbers. It is specified as `.` followed by a number.

- **`%.nf`**: Displays a floating-point number with **n** digits after the decimal point. The number will be rounded if necessary.

### Example Usage of Precision

```java
double price1 = 9.99;
double price2 = 100.15;
double price3 = -54.01;

// Default precision for %f is 6 decimal places
System.out.printf("%f\n", price1);
// Output: 9.990000

// Setting precision to 3 decimal places
System.out.printf("%.3f\n", price1);
// Output: 9.990

System.out.printf("%.3f\n", price2);
// Output: 100.150

System.out.printf("%.3f\n", price3);
// Output: -54.010
```

---

## Flags

Flags modify the output of a format specifier.

- **`+`**: Always prints a sign for a number, even if it's positive.
- **`,`**: Inserts a comma as a grouping separator for thousands.
- **`(`**: Encloses negative numbers in parentheses.
- **`(space)`**: Inserts a space before positive numbers and a minus sign before negative numbers.

### Example Usage of Flags

```java
double price1 = 9.99;
double price2 = 1000.15;
double price3 = -54.01;

// Default (%f)
System.out.printf("%f\n", price1); // Output: 9.990000
System.out.printf("%f\n", price2); // Output: 1000.150000
System.out.printf("%f\n", price3); // Output: -54.010000

// '+' flag
System.out.printf("%+f\n", price1); // Output: +9.990000
System.out.printf("%+f\n", price3); // Output: -54.010000

// ',' flag (for integers and floating-point numbers)
System.out.printf("%,.2f\n", price1); // Output: 9.99
System.out.printf("%,.2f\n", price2); // Output: 1,000.15
System.out.printf("%,.2f\n", price3); // Output: -54.01

// '(' flag for negative numbers
System.out.printf("(%f)\n", price1); // Output: (9.990000)
System.out.printf("(%f)\n", price3); // Output: (-54.010000)

// ' ' (space) flag
System.out.printf("'% f'\n", price1); // Output: '   9.990000' (space before positive)
System.out.printf("'% f'\n", price3); // Output: '-54.010000' (minus before negative)
```

---

## Width

The **width** specifier defines the minimum number of characters that the output will occupy. If the actual output is shorter than the specified width, it will be padded with spaces (by default).

- **`[width]`**: Right-justifies the output within the specified width.
- **`-[width]`**: Left-justifies the output within the specified width.
- **`0[width]`**: Zero-pads the output on the left if the number of digits is less than the specified width. This flag only works for numeric types.

### Example Usage of Width

```java
int id1 = 1;
int id2 = 23;
int id3 = 456;
int id4 = 7890;

// Default formatting for integers
System.out.printf("id: %d\n", id1); // Output: id: 1
System.out.printf("id: %d\n", id2); // Output: id: 23
System.out.printf("id: %d\n", id3); // Output: id: 456
System.out.printf("id: %d\n", id4); // Output: id: 7890

// Width of 4 (right-justified)
System.out.printf("id: %4d\n", id1); // Output: id:    1
System.out.printf("id: %4d\n", id2); // Output: id:   23
System.out.printf("id: %4d\n", id3); // Output: id:  456
System.out.printf("id: %4d\n", id4); // Output: id: 7890

// Width of 4 with zero-padding (04d)
System.out.printf("id: %04d\n", id1); // Output: id: 0001
System.out.printf("id: %04d\n", id2); // Output: id: 0023
System.out.printf("id: %04d\n", id3); // Output: id: 0456
System.out.printf("id: %04d\n", id4); // Output: id: 7890

// Negative width for left justification
System.out.printf("id: %-4d\n", id1); // Output: id: 1   
System.out.printf("id: %-4d\n", id2); // Output: id: 23  
System.out.printf("id: %-4d\n", id3); // Output: id: 456 
System.out.printf("id: %-4d\n", id4); // Output: id: 7890
```

---

## Newline Characters

Unlike `println()`, `printf()` does not automatically add a newline character at the end of the output. You must explicitly include `\n` in the format string when you want to move to the next line.

### Example

```java
String item = "Burger";
double price = 5.99;

System.out.printf("Item: %s\n", item); // Newline added
System.out.printf("Price: %.2f\n", price); // Newline added
// Output:
// Item: Burger
// Price: 5.99
```

# printf() Formatting Activities

## Activity 1: Receipt Generator
Create a program that generates a formatted receipt for a restaurant order. The program should:
1. Display items with names, quantities, and prices
2. Format all prices to 2 decimal places
3. Align columns using width specifiers
4. Include subtotal, tax (8.5%), and total
5. Add comma separators for amounts over $1,000

**Sample Output:**
```
==============================
        RESTAURANT RECEIPT
==============================
Item            Qty   Price   Total
----------------------------------
Burger            2   $5.99   $11.98
Fries             1   $2.49    $2.49
Soda              3   $1.99    $5.97
----------------------------------
Subtotal:                $20.44
Tax (8.5%):               $1.74
Total:                   $22.18
==============================
```

## Activity 2: Student Report Card
Create a program that displays a formatted student report card with:
1. Student information (name, ID, grade level)
2. Course list with grades (4-5 courses)
3. GPA calculation
4. Use width formatting to align columns
5. Format GPA to 3 decimal places
6. Display percentages with appropriate signs

**Bonus:** Add conditional formatting (like color codes using terminal escape sequences or simple text indicators) for grades below a certain threshold.

## Activity 3: Data Table Formatter
Create a program that generates a formatted data table from arrays of information. The table should include:
1. Column headers with consistent widths
2. Different data types (strings, integers, decimals)
3. Alignment appropriate to data type (left for text, right for numbers)
4. Currency formatting where applicable
5. Totals row at the bottom

**Example data:**
- Product names (strings)
- Units sold (integers)
- Price per unit (decimals)
- Total revenue (calculated, decimals)

**Challenge:** Make the program dynamic so it can handle different numbers of rows and still format correctly.
