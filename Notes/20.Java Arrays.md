# Java Arrays

## What is an Array?

An array in Java is a collection of values of the same data type. Think of it as a single variable that can store multiple values.

---

## Declaring and Initializing Arrays

You can declare and initialize an array in a few ways:

### 1. Declaration and Initialization in one step:

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
```

In this example, `fruits` is an array of strings. The values within the curly braces `{}` are the elements of the array.

### 2. Declaration and then Initialization:

```java
String[] colors; // Declaration
colors = new String[]{"red", "green", "blue"}; // Initialization
```

### 3. Declaring with a specific size:

```java
int[] numbers = new int[5]; // Declares an integer array that can hold 5 elements.
```

When an array is declared with a specific size, its elements are initialized to their default values (0 for numeric types, `false` for booleans, `null` for object types).

---

## Accessing Array Elements

Each value within an array is called an **element**. Elements are accessed using an **index number**.

- Array indices start at 0.
- The first element is at index 0, the second at index 1, and so on.

To access an element, you use the array name followed by square brackets `[]` containing the index:

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
System.out.println(fruits[0]); // Output: apple
System.out.println(fruits[1]); // Output: orange
```

## ArrayIndexOutOfBoundsException

If you try to access an index that does not exist in the array, you will get an `ArrayIndexOutOfBoundsException`.

For the `fruits` array above (which has 4 elements), the valid indices are 0, 1, 2, and 3. Accessing `fruits[4]` would cause this exception.

---

## Modifying Array Elements

You can change the value of an element at a specific index:

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
fruits[0] = "pineapple"; // Change the element at index 0
System.out.println(fruits[0]); // Output: pineapple
```

---

## Array Length

The `length` property of an array returns the number of elements it contains.

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
int numOfFruits = fruits.length; // numOfFruits will be 4
System.out.println(numOfFruits); // Output: 4
```

---

## Iterating Through Arrays

There are several ways to loop through all the elements in an array.

### Standard for Loop

A standard for loop uses an index variable to access each element.

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
for (int i = 0; i < fruits.length; i++) {
    System.out.println(fruits[i]);
}
// Output:
// apple
// orange
// banana
// coconut
```

In this loop:

- `int i = 0`: Initializes the index counter `i` to 0.
- `i < fruits.length`: The loop continues as long as `i` is less than the total number of elements in the `fruits` array.
- `i++`: Increments the index `i` by 1 after each iteration.

You can also print elements separated by spaces:

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
for (int i = 0; i < fruits.length; i++) {
    System.out.print(fruits[i] + " ");
}
// Output: apple orange banana coconut
```

### Enhanced for Loop (for-each loop)

The enhanced for loop provides a simpler way to iterate over collections, including arrays. It iterates once for each element in the array.

```java
String[] fruits = {"apple", "orange", "banana", "coconut"};
for (String fruit : fruits) {
    System.out.println(fruit);
}
// Output:
// apple
// orange
// banana
// coconut
```

In this loop:

- `String fruit`: Declares a variable `fruit` of the same type as the array elements. In each iteration, `fruit` will hold the current element.
- `: fruits`: Specifies the array to iterate over.

---

## Using the Arrays Class

Java provides a utility class `java.util.Arrays` that offers helpful methods for working with arrays. You need to import this class:

```java
import java.util.Arrays;
```

### Sorting Arrays

The `Arrays.sort()` method sorts the elements of an array. For strings, it sorts them alphabetically. For numbers, it sorts them in ascending order.

```java
String[] fruits = {"orange", "apple", "coconut", "banana"};
Arrays.sort(fruits);
System.out.println(Arrays.toString(fruits)); // Output: [apple, banana, coconut, orange]
```

### Filling Arrays

The `Arrays.fill()` method assigns a specified value to all elements in an array.

```java
String[] fruits = new String[4]; // Initialize an array of size 4
Arrays.fill(fruits, "pineapple");
System.out.println(Arrays.toString(fruits)); // Output: [pineapple, pineapple, pineapple, pineapple]
```

### Printing Arrays

The `Arrays.toString()` method is very useful for printing the contents of an array in a readable format.

```java
String[] fruits = {"apple", "orange", "banana"};
System.out.println(Arrays.toString(fruits)); // Output: [apple, orange, banana]
```

## Activity 2: Inventory Management System
Create an inventory system for a store using arrays:

**Inventory Structure:**
1. **Product Arrays:**
   - `String[] productNames`
   - `int[] productIDs`
   - `double[] prices`
   - `int[] quantities`
   - `String[] categories`

2. **Operations:**
   - Add new products
   - Update stock quantities
   - Process sales (reduce inventory)
   - Restock products
   - Search products by name/ID/category

3. **Reporting:**
   - List low stock items (quantity < threshold)
   - Calculate total inventory value
   - Generate sales reports
   - Sort products by price/quantity/name

**Example Features:**
```java
// Find all electronics with less than 10 in stock
for (int i = 0; i < productNames.length; i++) {
    if (categories[i].equals("Electronics") && quantities[i] < 10) {
        System.out.println(productNames[i] + " needs restocking!");
    }
}

// Calculate total value of inventory
double totalValue = 0;
for (int i = 0; i < productNames.length; i++) {
    totalValue += prices[i] * quantities[i];
}
```

**Advanced Challenge:**
- Implement a shopping cart system
- Add discount calculations
- Create purchase history tracking
